## Troubleshooting

### Issue: gh CLI not authenticated

**Solution:**
```bash
gh auth login
# Follow the prompts
```

### Issue: Can't push to GitHub

**Solution:**
```bash
# Check remote
git remote -v

# If using HTTPS, you may need to authenticate
# Consider switching to SSH:
git remote set-url origin git@github.com:amalik/BMAD-Enhanced.git
```

### Issue: Release already exists

**Solution:**
```bash
# Delete the release
gh release delete v1.0.3-alpha

# Delete the tag
git tag -d v1.0.3-alpha
git push origin :refs/tags/v1.0.3-alpha

# Try again
./create-github-release.sh
```
