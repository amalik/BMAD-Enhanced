## Key Insights

1. **Platform rendering is fundamentally different from terminal** — always verify on GitHub/npm before marking visual stories done
2. **`<div align="center"><pre>` causes per-line centering** — use fenced code blocks inside centered divs instead
3. **Emoji are 2 columns wide in monospace code blocks** — incompatible with fixed-width ASCII art boxes
4. **Figlet "ANSI Shadow" font has inherently unequal row widths** — rows 3-4 are 60 chars, others are 62. A different font or trailing space padding would fix this
5. **Simple stories (2.3) succeed when discovery is done upfront** — the verified workflow inventory in the story file meant zero surprises
